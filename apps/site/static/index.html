    How to use it
    
    
    
      
      
    1. Paste your text above, or drop in a PDF or Word document. Files are read on your device and never uploaded.
      2. 
      
    2. Press Mask. Identifying information lights up: names, schools, emails, phone numbers, and passages that might identify the student through context.
      3. 
      
    3. Review the highlights — click any one to approve, edit, or dismiss it. Caught something the scan missed? Select the text and press “Mask this,” or keep an always-flag list of names the scan should never miss. You know the student; the tool does not.
      4. 
      
    4. Copy the masked text and use it with any AI service you like.
      5. 
      
    5. When you get results back, paste them under Unmask. Real names return everywhere the AI kept the placeholders.
      6. 
      
    6. Working with the same student across a season? Export the mapping and import it next session — the same student is always “Student A,” across every document.
      7. 
    
    
    
    Three kinds of flags
    
    
      
      
    • j.rivera@example.org Direct identifiers — emails, phones, ID numbers, dates. Deterministic patterns, near-perfect reliability.
      • 
      
    • Maya Chen Names — people, organizations, places, detected by a small language model running on your own device, the way a spell-checker does. Nothing is sent anywhere.
      • 
      
    • first wrestling captain Contextual flags — passages with no names that may still identify the student. Flagged for your judgment, never masked automatically.

    How you can trust it
    
    
    
      
      
    1. Turn off your internet. Load the page, disconnect, and use the tool. It works fully offline because nothing it does requires a server.
      2. 
      
    2. Watch the network. Open your browser’s developer tools and observe that no request carries your text anywhere.
      3. 
      
    3. Read the code. The entire tool is open source under the MIT license.
      4. 
      
    4. Trust the incentives. This tool is free, has no accounts, and collects nothing. Its only value is its credibility.
      5. 
    
    
    
    One promise we will not make: NameMasker does not “guarantee anonymity.” No tool honestly can. It catches what software can catch, flags what software can only suspect, and puts you in the approval seat for everything. The review step is the product.
